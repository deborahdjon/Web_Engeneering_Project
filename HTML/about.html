Project name: Dynamic Calendar
Authors: Deborah Djon, Viet Dang

Description

Thy Dynamic Calendar is a web application is a project which was part of Web-Engineering 1 course of our university. It has all the basic and enhanced features listed below.
DHBW Stuttgart

Basic Fearures

  • Show calendar entries
  • Create calendar entry
  • Edit calendar entries
  • Delete calendar entries
  • Image for calendar entry

Enhanced Features

  • Alerts with sound (sound needs to be enabled by the user)
  • Back button handling on all pages
  • Show categories from server
  • Extra pages for editing and adding entries and categories
  • Image preview on the edit/add entry page
  • Refresh button handling on the edit/add entry page
  • About page
  • Contact page with contact form
